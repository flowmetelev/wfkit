# Changelog

All notable changes to `wfkit` will be documented in this file.

## 1.0.0

Initial public release.

Highlights:

- scaffold a new Webflow-ready Vite project with `wfkit init`
- develop against a published `.webflow.io` site through a local proxy
- publish global and page-based production bundles back to Webflow
- migrate legacy page and global custom code into local project files
- publish a markdown docs hub page into Webflow
- validate local setup with `wfkit doctor`
- check for updates, show richer TUI flows, and open issue forms from the CLI
